## 如何测试响应式效果

想知道页面在手机上看起来怎么样？不需要真的用手机，浏览器可以模拟：

1. 在浏览器中打开你的页面
2. 按 `F12` 打开开发者工具
3. 点击工具栏左上角的"设备切换"图标（长得像手机和平板）
4. 选择一个手机型号（如 iPhone 12）或者直接拖动页面宽度

检查这些情况：

- <input type="checkbox" name="如何测试响应式效果-checklist" value="页面内容没有超出屏幕" /> <label for="页面内容没有超出屏幕">页面内容没有超出屏幕</label>
- <input type="checkbox" name="如何测试响应式效果-checklist" value="文字大小在手机上也能看清" /> <label for="文字大小在手机上也能看清">文字大小在手机上也能看清</label>
- <input type="checkbox" name="如何测试响应式效果-checklist" value="按钮在手机上也容易点击" /> <label for="按钮在手机上也容易点击">按钮在手机上也容易点击</label>
- <input type="checkbox" name="如何测试响应式效果-checklist" value="输入框宽度适合手机屏幕" /> <label for="输入框宽度适合手机屏幕">输入框宽度适合手机屏幕</label>
