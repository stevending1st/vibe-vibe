## 常见问题速查

如果上面某项检查没通过，看看这里有没有解决方案：

### 问题：网络访问不稳定

**症状**：AI 回复超时，频繁断连

**解决方案**：
1. 尝试切换网络（如从 WiFi 切到手机热点）
2. 检查是否需要配置代理
3. 重启 AI IDE 后重试

### 问题：账号登录失败

**症状**：提示登录失败、账号不存在、密码错误

**解决方案**：
1. 确认使用正确的登录方式（邮箱/GitHub/Google 等）
2. 尝试重置密码
3. 检查邮箱是否收到验证邮件

### 问题：提示额度用完

**症状**：AI 不再回复，提示 "quota exceeded" 或类似信息

**解决方案**：
1. 等待月度额度刷新（通常每月 1 号）
2. 购买额外额度
3. 优化 Prompt，减少不必要的对话次数

### 问题：工具崩溃或卡住

**症状**：界面无响应，一直在加载

**解决方案**：
1. 重启 AI IDE
2. 检查是否有软件更新
3. 如果问题持续，尝试重新安装

### 问题：生成的代码看不懂

**症状**：AI 生成了很多代码，但你不知道是什么意思

**这不是问题**。Vibe Coding 的核心理念就是：你不需要完全理解代码。只要结果符合预期，代码细节可以交给 AI。

如果结果不符合预期，你需要做的是**告诉 AI 哪里不对**，而不是自己修改代码。

## 推荐的学习环境设置

为了获得最佳学习体验，建议：

### 屏幕布局

如果你有大屏幕或双显示器：
- 一边放 AI IDE（用于操作）
- 一边放本教程（用于参考）

如果只有一个屏幕：
- 可以用浏览器标签页切换
- 或者用分屏功能并排显示

### 窗口布局建议

开始之前，建议准备好：
1. **AI IDE**（全屏或半屏显示）
2. **浏览器**（打开本教程页面）

### 减少干扰

- 关闭不必要的通知
- 告诉家人/室友你需要一段专注时间
- 准备好水或饮料（减少起身次数）
