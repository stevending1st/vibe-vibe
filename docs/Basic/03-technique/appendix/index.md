# 附录：常用 Prompt 模板库

本附录提供一系列可直接复制使用的 Prompt 模板，覆盖项目创建、代码修改、问题解决和文档生成四大场景。

## 如何使用这份模板库

1. **找到你的场景**：根据下方索引表，快速定位适合你的模板
2. **复制模板**：将模板复制到你的 AI 对话窗口
3. **替换占位符**：把 `[方括号内容]` 替换成你的实际情况
4. **根据需要删减**：如果某些字段不适用，可以删除
