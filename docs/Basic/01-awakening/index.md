# 第 1 章：觉醒 —— 为什么现在是编程最好的时代

> **本章目标**：理解 AI 编程的时代背景，完成你的第一个 AI 网页。


2025 年，AI 编程工具让"用自然语言描述需求，AI 生成代码"成为现实。本章介绍这场变革的背景，并带你完成第一个 AI 网页。

## 本章你将学到

读完这一章，你会：

| 收获 | 说明 |
|------|------|
| **理解变革** | 明白为什么"编程"正在被重新定义 |
| **掌握概念** | 了解 Vibe Coding、Spec Coding 等核心概念 |
| **建立信心** | 打破"编程很难"的心理障碍 |
| **认识工具** | 知道有哪些 AI 编程工具可以用 |
| **动手实践** | 做出你人生中第一个 AI 网页 |

AI编程工具分为五大类：对话式AI、AI代码编辑器、IDE插件、网页版编程、命令行工具。本节提供详细的选型指南，帮助你根据自身情况选择最佳工具。

### 1.4 方法论：Vibe Coding与Spec Coding详解

选择了工具并完成初步实践后，需要深入理解AI编程的核心方法论。

Vibe Coding适用于需求模糊的探索阶段，Spec Coding适用于需求明确的执行阶段。掌握何时使用哪种方法，是高效AI编程的关键。

[开始阅读 →](./1.4-vibe-vs-spec/)
