# 1.3 工具初探：AI 编程工具选型指南

现在你已经知道编程没有想象中那么难，心理障碍也破除了。接下来，让我们选择合适的工具开始行动。

2025 年，AI 编程工具多得让人眼花缭乱：Cursor、Windsurf、Bolt.new、Trae、Kiro、v0.dev...

**别慌。** 这一章帮你理清思路，找到适合你的那一个。
