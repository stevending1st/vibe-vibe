# 0.2 命令行：与计算机的直接对话

欢迎来到命令行的世界！

如果说图形用户界面（GUI）是与计算机进行“菜单式”的互动，那么命令行界面（CLI）就是与计算机进行“对话式”的交流。你不再需要通过点击图标和按钮来操作，而是通过输入精确的文本指令，直接告诉计算机你想要做什么。

对于 Vibe Coder 来说，命令行不是一个过时的工具，而是一个高效、强大且可自动化的核心技能。在 AI 辅助编程的时代，我们更需要通过命令行来精准地控制开发环境、执行脚本、管理服务器。

在本章中，你将学到：

*   **文件系统导航**：如何在目录树中自由穿梭。
*   **文件基本操作**：如何创建、查看、移动和删除文件与目录。
*   **权限管理**：如何用命令行控制谁能动你的文件。
*   **跨平台差异**：理解 Windows (PowerShell) 和 macOS/Linux (Bash/Zsh) 之间的常用命令区别。

准备好，让我们开始学习这门计算机的“通用语言”。
