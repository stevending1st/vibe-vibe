## 摘要

这是一套围绕 Cursor 的零基础 vibe coding 入门教程，创作者 Tim 以“不是要把你训练成专业工程师，而是帮你用 AI 快速做出东西”为定位，核心关键词包括 Cursor、AI code generation、vibe coding workflow、version control、MCP tools。 视频从“什么是 vibe coding”讲起，展示如何安装与配置 Cursor、选取助手模型、规划项目，再通过对话式 prompt 生成应用、调试错误，过程中不断强调上下文管理、文档引用和规则设定，从而减少模型胡编乱改。 创新之处在于把复杂 IDE 能力拆成一条完整的初学者旅程：从 UI 讲解、制定计划、代码生成、调试，到使用 Git 保存历史和引入 MCP server 扩展工具链，让观众理解 vibe coding 不只是“聊天写代码”，而是包含规划、版本管理、扩展接口的一体化流程。 他也指出，若缺乏规则和测试，AI 生成代码很容易在规模放大后变得脆弱，因此建议在每一轮大改动后立即运行测试、记录决策，并通过自定义“Cursor 规则”给模型设定安全边界。
