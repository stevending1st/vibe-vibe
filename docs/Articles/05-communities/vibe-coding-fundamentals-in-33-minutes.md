## 摘要

这支 33 分钟教学由前 Meta 数据科学家 Tina Huang 主讲，聚焦如何把 vibe coding 落成一套“可复用的工作流”，而不是即兴乱试，关键词包括 vibe coding fundamentals、frameworks、Replit demo、Windsurf demo、software productivity。 她通过一份示例 PRD 展示如何用自然语言拆解需求、规划模块，再交给 Replit、Windsurf 等工具生成和迭代代码，并强调 Git 版本控制和系统化调试的重要性，以防止项目在频繁自动修改中失控。 创新点在于把 vibe coding 教成“框架课”：视频中总结了几条可操作原则，例如先写高层 PRD、再用逐步细化 prompt 指导 LLM、关键节点做人工 review 和小步测试，而不是一条 prompt 让模型“通关”；同时提供多个测验片段，帮助观众检验自己对错误排查和代码理解的掌握程度。 她也坦言，若完全把模型当黑箱，容易出现逻辑脆弱与可维护性问题，因此建议把 vibe coding 看成提升效率的“外骨骼”，而不是绕过编程思维的捷径。​
