## 摘要

文章系统梳理了使用 Anthropic Claude 进行 vibe coding 的核心理念与实践方法，强调这是一种意图驱动的编程范式，开发者通过自然语言描述期望的软件结果，关键词包括 intent-driven development、Anthropic Claude、AI-assisted coding、natural language programming。 作者指出 Claude 3.7 Sonnet 在理解复杂上下文和生成高质量代码方面的能力，使得即便没有编程经验的用户也能通过在 Windsurf 等 AI IDE 中定义规则来指导开发流程，从而大幅降低技术门槛。 文章的创新点在于将 vibe coding 与传统编程明确区分：传统方式要求开发者关注语法和算法细节，而 vibe coding 优先考虑功能意图与结果描述，AI 作为高级翻译器将自然语言转化为可执行代码，并通过持续的"描述-生成-审查-精炼"迭代循环来确保质量。​
