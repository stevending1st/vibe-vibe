## 摘要

微软官方发布的 GitHub Copilot vibe coding 工作坊材料，旨在教授开发者如何使用 Visual Studio Code 与 GitHub Copilot 进行高效的对话式开发，关键词包括 GitHub Copilot workshop、vibe coding training、Visual Studio Code、prompt engineering best practices。 文章指出，虽然简单提示如"给我一个露营装备市场应用"也能让 Copilot 交付结果，但往往与预期存在差距，因为 AI 会用自己的想象填补不确定性，因此工作坊强调提供清晰、详细的提示才是关键。 创新点在于微软将 vibe coding 作为正式培训内容推向开发者社区，提供结构化的练习场景、最佳实践案例和常见陷阱分析，帮助团队建立统一的 AI 协作规范，这也反映出大厂正在积极引导行业从"试探性使用"转向"规范化工程实践"。
